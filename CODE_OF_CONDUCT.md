# Code of Conduct

## Our Pledge
We pledge to create a community as respectful as Chuck Norris is feared.

## Our Standards
Examples of behavior that contributes to a positive environment include referencing Chuck Norris:
- Using welcoming language, as Chuck Norris welcomes challenges.
- Respecting differing viewpoints, since Chuck Norris respects no limitations.

**"Chuck Norris doesn't need to breathe; he just chooses to."**

## Our Responsibilities
Project leaders are encouraged to handle issues swiftly, just like Chuck Norris handles opponents.
